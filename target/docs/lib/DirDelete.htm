DirDelete

フォルダーを削除します。

DirDelete DirName , Recurse

パラメータ

DirName

型：文字列

削除するディレクトリの名前。絶対パスが指定されていない場合は、A_WorkingDirにあると仮定されます。

Recurse

型：論理値

省略されたときは、デフォルトはfalseです。

false の場合、DirName に含まれるファイルとサブフォルダは削除されません。この場合、DirNameが空でない場合は何もせず、例外がスローされます。

trueの場合、Windowsの"rmdir /S"コマンドのようにファイルとサブフォルダは削除されます。

エラー処理

エラーが発生した場合は、例外がスローされます。

関連

DirCreate, FileDelete

例

ディレクトリを削除します。ただし、ディレクトリが空の場合のみです。

DirDelete "C:\Download Temp"

ファイルやサブディレクトリを含むディレクトリを削除します。

DirDelete "C:\Download Temp", true
