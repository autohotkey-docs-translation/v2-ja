DirDelete

フォルダーを削除します。

DirDelete DirName , Recurse

パラメータ

DirName

型：文字列

削除するディレクトリの名前。絶対パスが指定されていない場合は、A_WorkingDirにあると仮定されます。

Recurse

型：論理値

This parameter specifies whether to recurse into subdirectories. If omitted, it defaults to false (no recursion). それ以外の場合は、次のいずれかの値を指定します：

0 または False:Do not remove files and sub-directories contained in DirName. In this case, if DirName is not empty, no action is taken and an exception is thrown.

1 または True:すべてのファイルとサブディレクトリを削除する（Windowsのコマンド"rmdir /S"のように）。

エラー処理

エラーが発生した場合は、例外が発生します。

関連

DirCreate, FileDelete

例

ディレクトリを削除します。ただし、ディレクトリが空の場合のみです。

DirDelete "C:\Download Temp"

ファイルやサブディレクトリを含むディレクトリを削除します。

DirDelete "C:\Download Temp", 1
