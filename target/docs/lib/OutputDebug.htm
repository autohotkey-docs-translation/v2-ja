OutputDebug

デバッガがある場合に文字列を送信して表示させます。

OutputDebug Text

パラメータ

Text

Type: String

デバッガに送信して表示させるテキストです。このテキストには、改行文字(`n)を含めることができます。また、1本の長い線を、継続部によっていくつかの短い線に分割することができます。

備考

スクリプトのプロセスにデバッガがない場合、システムデバッガが文字列を表示します。システムデバッガがアクティブでない場合、この機能は何の効果もありません。

デバッガの一例として、DebugViewがあり、microsoft.comで無償配布されています。

See also: other debugging methods

関連

FileAppend, continuation sections

例

デバッガがある場合に文字列を送信して表示させます。

OutputDebug A_Now ': Because the window "' TargetWindowTitle '" did not exist, the process was aborted.'
