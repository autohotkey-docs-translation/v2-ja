FileRecycle

ファイルやディレクトリを可能な限りごみ箱に送ったり、永久に削除したりします。

FileRecycle FilePattern

パラメータ

FilePattern

型：文字列

単一のファイル名、または "C:\Temp\*.tmp" のようなワイルドカードパターンを指定します。FilePatternは、絶対パスが指定されない場合、A_WorkingDirにあるとみなされます。

アスタリスク（*）とクエスチョンマーク（?）は、ワイルドカードとして使用できます。*は0以上の文字と、クエスチョンマークは?1文字とマッチします。使用例：

  • *.*と*は、全てのファイルにマッチします。
  • *.htmは、.htmや.htmlなどの拡張子を持つファイルにマッチします。
  • *.は、拡張子がないファイルにマッチします。
  • log?.txtは、例えばlog1.txtにはマッチしますが、log10.txtにはマッチしません。
  • *report*は、ファイル名に"report"を含むあらゆるファイルにマッチします。

ディレクトリ全体をリサイクルする場合は、末尾のバックスラッシュを除いた名前を指定します。

エラー処理

失敗した場合は例外がスローされます。

備考

実際の作業にはSHFileOperationが使用されます。この関数は、ファイルが大きすぎてリサイクルできない場合、ファイルを永久に削除することがあります。

など、他の理由でファイルをリサイクルできない場合、警告なしにファイルを永久に削除することがあります：

  • リムーバブルドライブ上にあるファイルです。
  • NukeOnDeleteなどのレジストリ値によってごみ箱は無効化されます。

関連

FileRecycleEmpty、FileDelete、FileCopy、FileMove

例

ディレクトリ内のすべての .tmp ファイルを可能な限りごみ箱に送ります。

FileRecycle "C:\temp files\*.tmp"
