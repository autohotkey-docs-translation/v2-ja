SoundSetMute

サウンドデバイスのミュート設定を変更します。

SoundSetMute NewSetting , Component, Device

パラメータ

NewSetting

型：数字または文字列

正数ならオン（ミュート）、ゼロならオフ（ミュート解除）。ただし、数字がプラス記号またはマイナス記号で始まる場合、その設定はトグルされる（現在の状態とは逆の状態に設定される）。

コンポーネント

型：文字列または整数

コンポーネントの表示名とインデックス。例えば、1、"Line in"または"Line in:2"。

省略または空白の場合、マスターミュートの設定が変更される。

詳しくは「コンポーネント（サウンド機能）」をご覧ください。

Device

型：文字列または整数

デバイスの表示名とインデックス。例えば、1、"Speakers"、"Speakers:2"、"Speakers (Example HD Audio)"などです。

このパラメーターが省略された場合、デフォルトはシステムのデフォルト再生デバイス（デバイス1とは限らない）となる。

詳しくはデバイス（サウンド機能）をご覧ください。

エラー処理

失敗すると以下の例外のいずれかがスローされる：

エラークラスメッセージ
TargetErrorデバイスが見つかりません
コンポーネントが見つかりません
コンポーネントがこのコントロールタイプをサポートしていない
OSErrorシステムが生成する。

備考

デフォルトの再生デバイスのマスターミュート設定を切り替える別の方法は、以下の例のように、スクリプトにキーストロークを送信させることです：

Send "{Volume_Mute}"  ; Mute/unmute the master volume.

システムにインストールされているサウンドデバイスの機能（名前や使用可能なコンポーネントなど）を調べるには、サウンドカード解析スクリプトを実行します。

現在のミュート設定を取得するには、SoundGetMuteを使います。

関連

サウンド機能

例

マスターミュートを設定する（オン）。

SoundSetMute true

マスターミュートを切り替える（逆の状態にする）。

SoundSetMute -1

ミュート・ライン・イン

SoundSetMute 1, "Line In"

マイク録音をミュートする。

SoundSetMute 1, , "Microphone"
