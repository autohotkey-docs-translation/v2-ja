Float

数値文字列または整数を浮動小数点数に変換します。

FltValue := Float(Value)

戻り値

型：浮動小数点数

この関数は、Valueを純粋な浮動小数点数（型名"Float"）に変換した結果、またはValueが既に正しい型である場合はValue自身を返す。

備考

値を変換できない場合、TypeErrorがスローされます。

ある値が浮動小数点数に変換できるかどうかを調べるには、IsNumber関数を使用します。

Floatは実際にはクラスですが、関数として呼び出すことができます。Value is Floatは、値が純粋な浮動小数点数であるかどうかをチェックするために使用できる。
