DriveGetList

システム内の各ドライブレターに対して1文字ずつ、文字列を返します。

List := DriveGetList(DriveType)

パラメータ

DriveType

Type: String

省略した場合は、すべてのドライブタイプを取得します。Otherwise, specify one of the following words to retrieve only a specific type of drive: CDROM, REMOVABLE, FIXED, NETWORK, RAMDISK, UNKNOWN.

戻り値

Type: String

This function returns the drive letters in the system, depending on DriveType. For example: ACDEZ.
