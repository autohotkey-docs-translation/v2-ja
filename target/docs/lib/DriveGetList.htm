DriveGetList

システム内の各ドライブレターに対して1文字ずつ、文字列を返します。

List := DriveGetList(DriveType)

パラメータ

DriveType

型：String

省略した場合は、すべてのドライブタイプを取得します。それ以外の場合は、次のいずれかの単語を指定して、特定の種類のドライブのみを検索します：CDROM, REMOVABLE, FIXED, NETWORK, RAMDISK, UNKNOWN.

戻り値

型：String

This function returns the drive letters in the system, depending on DriveType. 事例：ACDEZ.
