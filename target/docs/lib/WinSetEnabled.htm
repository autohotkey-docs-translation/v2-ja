WinSetEnabled

指定したウィンドウを有効または無効にします。

WinSetEnabled Value , WinTitle, WinText, ExcludeTitle, ExcludeText

パラメータ

値

型：整数

以下のいずれかの値：

  • 1またはTrueは設定をオンにする
  • 0またはFalseは設定をオフにする
  • -1では、現在の状態とは逆の状態に設定されます
WinTitle

型：文字列、整数、またはオブジェクト

ウィンドウのタイトルなど、対象のウィンドウを特定する条件。WinTitleを参照してください。

WinText

型：文字列

存在する場合、このパラメータは、ターゲットウィンドウの単一のテキスト要素からの部分文字列でなければなりません（付属のWindow Spyユーティリティによって明らかにされます）。DetectHiddenTextがONの場合、隠しテキスト要素が検出されます。

ExcludeTitle

型：文字列

タイトルにこの値が含まれているWindowsは、審査対象外となります。

ExcludeText

型：文字列

テキストにこの値が含まれているウィンドウは、検討の対象外となります。

エラー処理

A TargetError is thrown if the window could not be found.

An OSError is thrown if the change could not be applied.

備考

When a window is disabled, the user cannot move it or interact with its controls. In addition, disabled windows are omitted from the alt-tab list.

Example #1 on the WinGetStyle page can be used to determine whether a window is disabled.

The ID of the window under the mouse cursor can be retrieved with MouseGetPos.

ウィンドウのタイトルとテキストは大文字と小文字が区別されます。DetectHiddenWindowsがオンになっていない限り、隠しウィンドウは検出されません。
