WinShow

指定したウィンドウの非表示を解除します。

WinShow WinTitle, WinText, ExcludeTitle, ExcludeText

パラメータ

WinTitle、WinText、ExcludeTitle、ExcludeText

型：文字列、整数、またはオブジェクト

If each of these is blank or omitted, the Last Found Window will be used. そうでない場合は、WinTitleに ウィンドウタイトルまたはターゲットウィンドウを識別するための他の基準、および/またはWinTextにターゲットウィンドウの単一のテキスト要素からの部分文字列（同梱のWindow Spyユーティリティによって明らかにされる）を指定します。

ExcludeTitle and ExcludeText can be used to exclude one or more windows by their title or text. Their specification is similar to WinTitle and WinText, except that ExcludeTitle does not recognize any criteria other than the window title.

ウィンドウのタイトルとテキストは大文字と小文字を区別します。By default, hidden text elements are detected unless changed with DetectHiddenText. By default, a window title can contain WinTitle or ExcludeTitle anywhere inside it to be a match, unless changed with SetTitleMatchMode.

エラー処理

グループモードが使用されている場合を除き、ウィンドウが見つからなかった場合はTargetErrorがスローされる。

備考

デフォルトでは、WinShowは常に隠しウィンドウを検出できる唯一の機能です。他の組み込み関数は、DetectHiddenWindowsがオンになっている場合のみ検出できる。

ただし、WinTitleがahk_group GroupNameの場合は、グループ内のすべてのウィンドウが対象となります。

関連

WinHide, SetTitleMatchMode, DetectHiddenWindows, Last Found Window

例

メモ帳を開き、メモ帳が存在するまで待ち、短時間非表示にし、非表示を解除する。

Run "notepad.exe"
WinWait "Untitled - Notepad"
Sleep 500
WinHide ; Use the window found by WinWait.
Sleep 1000
WinShow ; Use the window found by WinWait.
