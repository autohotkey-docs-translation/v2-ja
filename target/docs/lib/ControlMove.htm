ControlMove

コントロールを移動またはサイズ変更します。

ControlMove X, Y, Width, Height, Control, WinTitle, WinText, ExcludeTitle, ExcludeText

パラメータ

X, Y

型：整数

コントロールの新しい位置の左上隅のX、Y座標（ピクセル単位）。どちらかの座標が省略された場合、その次元におけるコントロールの位置は変更されません。座標は、ターゲットウィンドウのクライアント領域の左上隅からの相対座標で、ControlGetPosで決定できます。

幅、高さ

型：整数

Controlの新しい幅と高さ（ピクセル単位）です。どちらかのパラメータが省略された場合、その次元におけるコントロールのサイズは変更されません。

Control

型：文字列、整数、またはオブジェクト

コントロールのClassNN、text、HWND、またはHwndプロパティを持つオブジェクトを指します。詳しくは、「制御パラメータ」をご覧ください。

WinTitle

型：文字列、整数、またはオブジェクト

ウィンドウのタイトルなど、対象のウィンドウを特定する条件。WinTitleを参照してください。

WinText

型：文字列

存在する場合、このパラメータは、ターゲットウィンドウの単一のテキスト要素からの部分文字列でなければなりません（付属のWindow Spyユーティリティによって明らかにされます）。DetectHiddenTextがONの場合、隠しテキスト要素が検出されます。

ExcludeTitle

型：文字列

タイトルにこの値が含まれているWindowsは、審査対象外となります。

ExcludeText

型：文字列

テキストにこの値が含まれているウィンドウは、検討の対象外となります。

エラー処理

ウィンドウまたはコントロールが見つからない場合、TargetErrorが投げられます。

コントロールの現在位置を特定できなかった場合、OSErrorがスローされます。

備考

信頼性向上のため、本機能を使用するたびに自動的に遅延が行われます。この遅延はSetControlDelayで変更することができます。

ウィンドウのタイトルとテキストは大文字と小文字が区別されます。DetectHiddenWindowsがオンになっていない限り、隠しウィンドウは検出されません。

関連

ControlGetPos, WinMove, SetControlDelay, Control functions

例

スクリプトがユーザーの入力を待っている間に、入力ボックスのOKボタンを操作する方法を説明します。

SetTimer ControlMoveTimer
IB := InputBox(, "My Input Box")

ControlMoveTimer()
{
    if !WinExist("My Input Box")
        return
    ; それ以外の場合は、"last found "ウィンドウを設定します。
    SetTimer , 0
    WinActivate
    ControlMove 10,, 200,, "OK"  ; OKボタンを左側に移動し、幅を広げる。
}
