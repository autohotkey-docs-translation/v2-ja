EnvGet

環境変数を取得します。

Value := EnvGet(EnvVarName)

パラメータ

EnvVarName

型：文字列

取得する環境変数名。事例：Path := EnvGet("Path").

戻り値

型：文字列

説明： 指定された環境変数の値を返します。指定された環境変数が空または存在しない場合は、空の文字列が返されます。

備考

オペレーティングシステムは、各環境変数のテキストを32KBに制限しています。

通常のスクリプト変数は環境に保存されないため、この機能が存在します。これは、パフォーマンスが悪くなることと、OSが環境変数を32KBに制限しているためです。

関連

EnvSet, environment variables, Run, RunWait

例

環境変数を取得し、その値をLogonServerに格納する。

LogonServer := EnvGet("LogonServer")

Program Files」ディレクトリのパスを取得し、報告します。別の方法として、RegReadの例 #2を参照してください。

ProgramFilesDir := EnvGet(A_Is64bitOS ? "ProgramW6432" : "ProgramFiles")
MsgBox "Program files are in: " ProgramFilesDir

現在のユーザーのローカルAppDataディレクトリのパスを取得し、報告します。

LocalAppData := EnvGet("LocalAppData")
MsgBox A_UserName "'s Local directory is located at: " LocalAppData
