EditGetLine

Edit コントロールの指定された行のテキストを返します。

Line := EditGetLine(N, Control , WinTitle, WinText, ExcludeTitle, ExcludeText)

パラメータ

N

型：整数

The line number. Line 1 is the first line.

Control

型：文字列、整数、またはオブジェクト

コントロールのClassNN、text、HWND、またはHwndプロパティを持つオブジェクトを指します。詳しくは、「制御パラメータ」をご覧ください。

WinTitle

型：文字列、整数、またはオブジェクト

ウィンドウのタイトルなど、対象のウィンドウを特定する条件。WinTitleを参照してください。

WinText

型：文字列

存在する場合、このパラメータは、ターゲットウィンドウの単一のテキスト要素からの部分文字列でなければなりません（付属のWindow Spyユーティリティによって明らかにされます）。DetectHiddenTextがONの場合、隠しテキスト要素が検出されます。

ExcludeTitle

型：文字列

タイトルにこの値が含まれているWindowsは、審査対象外となります。

ExcludeText

型：文字列

テキストにこの値が含まれているウィンドウは、検討の対象外となります。

戻り値

型：文字列

This function returns the text of line N in an Edit control. Depending on the nature of the control, the string might end in a carriage return (`r) or a carriage return + linefeed (`r`n).

エラー処理

ウィンドウまたはコントロールが見つからない場合、TargetErrorが投げられる。

A ValueError is thrown if N is out of range or otherwise invalid.

コントロールにメッセージを送信できなかった場合、OSErrorがスローされます。

備考

ウィンドウのタイトルとテキストは大文字と小文字が区別されます。DetectHiddenWindowsがオンになっていない限り、隠しウィンドウは検出されません。

関連

EditGetCurrentCol, EditGetCurrentLine, EditGetLineCount, EditGetSelectedText, EditPaste, Control functions

例

Retrieves the first line of the Notepad's Edit control.

line1 := EditGetLine(1, "Edit1", "ahk_class Notepad")
