EditPaste

編集コントロールのキャレット（テキスト挿入点）に指定された文字列を貼り付けます。

EditPaste String, Control , WinTitle, WinText, ExcludeTitle, ExcludeText

パラメータ

String

型：文字列

The string to paste.

Control

型：文字列、整数、またはオブジェクト

コントロールのClassNN、text、HWND、またはHwndプロパティを持つオブジェクトを指します。詳しくは、「制御パラメータ」をご覧ください。

WinTitle

型：文字列、整数、またはオブジェクト

ウィンドウのタイトルなど、対象のウィンドウを特定する条件。WinTitleを参照してください。

WinText

型：文字列

存在する場合、このパラメータは、ターゲットウィンドウの単一のテキスト要素からの部分文字列でなければなりません（付属のWindow Spyユーティリティによって明らかにされます）。DetectHiddenTextがONの場合、隠しテキスト要素が検出されます。

ExcludeTitle

型：文字列

タイトルにこの値が含まれているWindowsは、審査対象外となります。

ExcludeText

型：文字列

テキストにこの値が含まれているウィンドウは、検討の対象外となります。

エラー処理

ウィンドウまたはコントロールが見つからない場合、TargetErrorが投げられる。

An OSError may be thrown if a message could not be sent to the control.

備考

The effect is similar to pasting by pressing Ctrl+V, but this function does not affect the contents of the clipboard or require the control to have the keyboard focus.

信頼性向上のため、本機能を使用するたびに自動的に遅延が行われます。この遅延は、SetControlDelayまたはA_ControlDelayに値を割り当てることで変更可能です。詳しくは、SetControlDelayの備考を参照してください。

ウィンドウのタイトルとテキストは大文字と小文字が区別されます。DetectHiddenWindowsがオンになっていない限り、隠しウィンドウは検出されません。
