DetectHiddenText

ウィンドウ内の不可視テキストが、そのウィンドウを見つけるために「見える」かどうかを判定します。これはWinExist関数やWinActivate関数のようなウィンドウ関連の関数に影響します。

DetectHiddenText Mode

パラメータ

Mode

Type: Boolean

trueのときは、不可視テキストを検出します。

falseのときは、不可視テキストを検出しません。<

戻り値

Type: Integer (boolean)

この関数は、前の設定を返します。0（false）または1（true）のいずれかです。

備考

DetectHiddenText関数を使わなかったときは、初期値は1（true）です。

"隠しテキスト "とは、ウィンドウのコントロールのうち、表示されていないものを指す言葉です。そのため、そのテキストは「非表示」とされています。DetectHiddenTextをオフにすると、マルチペインウィンドウやマルチタブダイアログの異なるペイン間の違いを検出したい場合に便利です。Window Spyを使用して、現在アクティブなウィンドウのどのテキストが隠されているかを判断します。WinTextパラメータを受け付けるすべての組み込み関数（WinActivate、WinActive、WinWait、WinExistを含む）は、この設定の影響を受けます。

組込変数A_DetectHiddenTextに現在の設定値（1または0）が格納されます。

新しく起動するスレッド（ホットキー、カスタムメニュー項目、時間指定サブルーチンなど）はすべて、この機能のデフォルト設定で新しくスタートします。このデフォルトは、スクリプト起動時にこの関数を使用することで変更することができます。

関連

例

隠しテキストの検出をオフにします。

DetectHiddenText false
