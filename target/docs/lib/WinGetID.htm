WinGetID

指定されたウィンドウのユニークなID番号を返します。

ID := WinGetID(WinTitle, WinText, ExcludeTitle, ExcludeText)

パラメータ

WinTitle、WinText、ExcludeTitle、ExcludeText

型：文字列、整数、またはオブジェクト

これらの各項目が未設定または省略された場合、最後に見つかったウィンドウが使用される。そうでない場合は、WinTitleに ウィンドウタイトルまたはターゲットウィンドウを識別するための他の基準、および/またはWinTextにターゲットウィンドウの単一のテキスト要素からの部分文字列（同梱のWindow Spyユーティリティによって明らかにされる）を指定します。隠しテキスト要素は、DetectHiddenTextがオンになっている場合に検出されます。タイトルまたはテキストにExcludeTitleまたはExcludeTextが含まれているウィンドウは考慮されません。

戻り値

型：整数

この関数は、指定されたウィンドウのウィンドウハンドル（HWND）を返します。

エラー処理

ウィンドウが見つからない場合は、TargetErrorがスローされます。

備考

この関数はWinExistと同等である。

ウィンドウのID番号は、そのウィンドウが生きている間だけ有効である。言い換えれば、アプリケーションが再起動すれば、そのウィンドウはすべて新しいID番号を持つことになる。

マウスカーソルの下にあるウィンドウのIDは、MouseGetPosで取得できる。コントロールのHWNDを検出するには（PostMessage、SendMessage、またはDllCallで使用するため）、ControlGetHwndまたはMouseGetPosを使用します。

ウィンドウのタイトルとテキストは大文字と小文字を区別します。DetectHiddenWindowsがオンになっていない限り、隠しウィンドウは検出されません。

関連

WinGetIDLast, ControlGetHwnd, Hwnd property (Gui object), Win functions, Control functions

例

アクティブウィンドウを最大化し、一意のIDを報告する。

active_id := WinGetID("A")
WinMaximize active_id
MsgBox "The active window's ID is " active_id
