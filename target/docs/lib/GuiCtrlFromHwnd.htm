GuiCtrlFromHwnd

指定された HWND に関連付けられた GUI コントロールのGuiControl オブジェクトを取得します。

GuiControlObj := GuiCtrlFromHwnd(Hwnd)

パラメータ

Hwnd

型：整数

GUIコントロールのウィンドウハンドル（HWND）、またはその子ウィンドウ（例：ComboBoxのEditコントロール）を指します。コントロールは、Add またはAddControlTypeを呼び出して、現在のスクリプトによって作成されている必要があります。

戻り値

型：GuiControlまたはString(空)

この関数は、指定された HWND に関連付けられたGuiControl オブジェクト、または、存在しないか HWND が無効である場合は空文字列を返します。

備考

例えば、GUIコントロールのHWNDは、GuiCtrl.Hwnd、MouseGetPos、OnMessageで取得することができます。
