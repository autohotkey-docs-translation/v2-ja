GuiCtrlFromHwnd

指定されたウィンドウハンドルに関連付けられているGUIコントロールのGuiControlオブジェクトを取得します。 

GuiControlObj := GuiCtrlFromHwnd(Hwnd)

パラメータ

Hwnd

Type: Integer

GUIコントロールのウィンドウハンドル（HWND）、またはその子ウィンドウ（例：ComboBoxのEditコントロール）を指します。コントロールは、MyGui.Addで作成されている必要があります。

戻り値

Type: Object or String (empty)

この関数は、指定された HWND に関連付けられたGuiControl オブジェクト、または、存在しないか HWND が無効である場合は空文字列を返します。

備考

例えば、GUIコントロールのHWNDは、GuiCtrl.Hwnd、MouseGetPos、OnMessageで取得することができます。
