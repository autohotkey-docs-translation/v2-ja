GuiCtrlFromHwnd

Retrieves the GuiControl object of a GUI control associated with the specified window handle.

GuiControlObj := GuiCtrlFromHwnd(Hwnd)

パラメータ

Hwnd

型：整数

GUIコントロールのウィンドウハンドル（HWND）、またはその子ウィンドウ（例：ComboBoxのEditコントロール）を指します。The control must have been created by the current script via MyGui.Add.

戻り値

型：Object or String (empty)

この関数は、指定された HWND に関連付けられたGuiControl オブジェクト、または、存在しないか HWND が無効である場合は空文字列を返します。

備考

例えば、GUIコントロールのHWNDは、GuiCtrl.Hwnd、MouseGetPos、OnMessageで取得することができます。

関連
