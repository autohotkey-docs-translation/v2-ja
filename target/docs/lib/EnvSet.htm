EnvSet

環境に含まれる変数に値を書き込む。

EnvSet EnvVar , Value

パラメータ

EnvVar

型：文字列

使用する環境変数名（例："COMSPEC"、"PATH"）。

値

型：文字列

環境変数に設定する値。省略した場合は、環境変数を削除します。

エラー処理

失敗するとOSErrorが投げられる。

備考

オペレーティングシステムは、各環境変数のテキストを32KBに制限しています。

この機能で作成・変更された環境変数は、スクリプトがRunまたはRunWaitで起動したプログラムのみがアクセスできるようになります。詳しくは、環境変数をご覧ください。

通常のスクリプト変数は環境に保存されないため、この機能が存在します。これは、パフォーマンスが悪くなることと、OSが環境変数を32KBに制限しているためです。

関連

EnvGet, environment variables, Run, RunWait

例

環境に含まれるAutGUI変数にテキストを書き込む。

EnvSet "AutGUI", "Some text to put in the variable."
