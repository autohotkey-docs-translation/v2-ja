ControlChooseIndex

リストボックス、コンボボックス、タブコントロールの選択範囲を、指定された項目またはタブ番号に設定します。

ControlChooseIndex N, Control , WinTitle, WinText, ExcludeTitle, ExcludeText

パラメータ

N

型：整数

エントリまたはタブのインデックスで、1は最初のエントリまたはタブ、2は2番目のエントリまたはタブ、など。ListBoxやComboBoxの全項目を非選択にする場合は、0を指定します。

Control

型：文字列、整数、またはオブジェクト

コントロールのClassNN、text、HWND、またはHwndプロパティを持つオブジェクトを指します。詳しくは、「制御パラメータ」をご覧ください。

WinTitle

型：文字列、整数、またはオブジェクト

ウィンドウのタイトルなど、対象のウィンドウを特定する条件。WinTitleを参照してください。

WinText

型：文字列

存在する場合、このパラメータは、ターゲットウィンドウの単一のテキスト要素からの部分文字列でなければなりません（付属のWindow Spyユーティリティによって明らかにされます）。DetectHiddenTextがONの場合、隠しテキスト要素が検出されます。

ExcludeTitle

型：文字列

タイトルにこの値が含まれているWindowsは、審査対象外となります。

ExcludeText

型：文字列

テキストにこの値が含まれているウィンドウは、検討の対象外となります。

エラー処理

ウィンドウまたはコントロールが見つからない場合、またはコントロールのクラス名に "Combo"、"List"、"Tab "が含まれない場合、TargetErrorが発生します。

変更を適用できなかった場合、ErrorまたはOSErrorが投げられる。

備考

マルチセレクトリストボックスのすべての項目を選択する場合は、次のようにします。

PostMessage(0x0185, 1, -1, "ListBox1", WinTitle)  ; リストボックスの全項目を選択する。0x0185はLB_SETSELです。

信頼性向上のため、本機能を使用するたびに自動的に遅延が行われます。この遅延は、SetControlDelayまたはA_ControlDelayに値を割り当てることで変更可能です。詳しくは、SetControlDelayの備考を参照してください。

ウィンドウのタイトルとテキストは大文字と小文字が区別されます。DetectHiddenWindowsがオンになっていない限り、隠しウィンドウは検出されません。
