#MaxThreads

最大同時スレッド数を設定します。

#MaxThreads Value

パラメータ

Value

Type: Integer

同時に存在できるスレッドの最大総数です。255より大きい数値を指定しても、255を指定したのと同じになります。

備考

このディレクティブがスクリプトで指定されていないときは、10に設定されているとみなされます。

この設定はスクリプト全体の動作に影響を与えるため、スクリプトの任意の場所で1回だけ指定する必要があります。

1という値も可能ですが、スクリプトがメッセージボックスや、その他のダイアログを表示しているときに、新しいホットキーが起動しなくなるため、お勧めしません。また、他のスレッドがスリープや待機しているときにタイマーが作動するのを防ぐことができます。

OnExitコールバック関数は、スレッドがいくつ存在しても常に起動します。

この設定が#MaxThreadsPerHotkeyよりも低い場合、事実上その設定を上書きします。

他のディレクティブと同様に、#MaxThreadsは条件付きで実行することはできません。

関連

#MaxThreadsPerHotkey、スレッド、A_MaxHotkeysPerInterval、ListHotkeys

例

最大10スレッドではなく、最大2スレッドを可能にします。

#MaxThreads 2
