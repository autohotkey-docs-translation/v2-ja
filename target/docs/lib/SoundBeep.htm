SoundBeep

PCのスピーカーから音色を出します。

SoundBeep Frequency, Duration

パラメータ

Frequency

型：整数

If omitted, it defaults to 523. それ以外の場合は、音の周波数を37から32767の間で指定する。

Duration

型：整数

If omitted, it defaults to 150. それ以外の場合は、音の長さをミリ秒単位で指定する。

備考

スクリプトは、音が鳴り終わるのを待ってから続行する。さらに、音作りの際にシステムの応答性が低下する可能性もある。

コンピュータにサウンドカードがない場合、標準的なビープ音がPCのスピーカーから再生されます。

PCスピーカーをビープ音で鳴らす代わりに標準のシステム音を出すには、SoundPlayのアスタリスクモードを参照してください。
