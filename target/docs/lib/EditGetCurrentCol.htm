EditGetCurrentCol

キャレット（テキスト挿入点）が存在するエディットコントロールのカラム番号を返します。

CurrentCol := EditGetCurrentCol(Control , WinTitle, WinText, ExcludeTitle, ExcludeText)

パラメータ

Control

型：文字列、整数、またはオブジェクト

コントロールのClassNN、text、HWND、またはHwndプロパティを持つオブジェクトを指します。詳しくは、「制御パラメータ」をご覧ください。

WinTitle

型：文字列、整数、またはオブジェクト

ウィンドウのタイトルなど、対象のウィンドウを特定する条件。WinTitleを参照してください。

WinText

型：文字列

存在する場合、このパラメータは、ターゲットウィンドウの単一のテキスト要素からの部分文字列でなければなりません（付属のWindow Spyユーティリティによって明らかにされます）。DetectHiddenTextがONの場合、隠しテキスト要素が検出されます。

ExcludeTitle

型：文字列

タイトルにこの値が含まれているWindowsは、審査対象外となります。

ExcludeText

型：文字列

テキストにこの値が含まれているウィンドウは、検討の対象外となります。

戻り値

型：整数

This function returns the column number in an Edit control where the caret (text insertion point) resides. The first column is 1. If there is text selected in the control, the return value is the column number where the selection begins.

エラー処理

An exception is thrown on failure, or if the window or control could not be found.

備考

ウィンドウのタイトルとテキストは大文字と小文字が区別されます。DetectHiddenWindowsがオンになっていない限り、隠しウィンドウは検出されません。
