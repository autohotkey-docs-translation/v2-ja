#WinActivateForce

ウィンドウをアクティブにする穏やかな方法をスキップして、強引な方法へと直行します。

#WinActivateForce

このディレクティブをスクリプトの任意の場所に記述すれば、WinActivate、WinActivateBottom、GroupActivateなどのウィンドウをアクティブにする組み込み関数が、ウィンドウをアクティブにする「優しい」方法をスキップして、より強力な方法を直行するようになります。

このディレクティブは、通常、ウィンドウの起動の速さや信頼性を変えることはありませんが、異なるウィンドウを次々に起動したときに、タスクバーのボタンが点滅するのを防ぐことができます。

備考

他のディレクティブと同様に、#WinActivateForceを条件付きで実行することはできません。

関連

WinActivate、WinActivateBottom、GroupActivate

例

ウィンドウをアクティブにする強制的な方法を有効にします。

#WinActivateForce
