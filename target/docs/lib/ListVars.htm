ListVars

スクリプトの変数を表示します：その名前と現在の内容です。

ListVars

備考

This function is equivalent to selecting the "View->Variables" menu item in the main window. It can help debug a script.

リスト内の各変数について、変数の名前と内容が表示され、変数が含む内容に応じて他の情報も表示されます。各項目はキャリッジリターンとニューライン（`r`n）で終了するが、変数に`r`nが含まれている場合は複数行に渡ることもあります。

リスト項目は次のような形式をとることができる（イタリック体の単語はプレースホルダー）：

VarName[容量の長さ]：String
VarName:TypeName object {Info}
VarName:Number

Capacityは、その変数の現在の容量です。

Stringは、変数の文字列値の最初の60文字です。

情報はオブジェクトの種類によって異なりますが、現在は非常に限られています。

ListVarsを関数内で使用した場合、以下のようになります：

  • ローカル変数（現在の関数が参照する外部関数の変数を含みます）。
  • 現在の関数の静的変数です。関数内部で宣言されたグローバル変数も、このセクションにリストアップされています。
  • 現在の関数が他の関数の中にネストされている場合、各外部関数の静的変数もリストアップされます。
  • すべてのグローバル変数。

関連
