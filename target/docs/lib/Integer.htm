Integer

数値文字列または浮動小数点値を整数に変換します。

IntValue := Integer(Value)

戻り値

型：整数

この関数は、Valueを純粋な整数（型名「Integer」）に変換した結果、またはValueが、すでに正しい型である場合はそれ自体を返します。

備考

Valueの端数部分はすべて取り除かれ、Value < 0 ? Ceil(Value) : Floor(Value)と同じになる。

値を変換できない場合、TypeErrorがスローされます。

ある値が整数に変換できるかどうかを判断するには、IsNumber関数を使用します。

Integerは実際にはクラスですが、関数として呼び出すことができます。Value is Integerは、値が純粋な整数であるかどうかをチェックするために使用できます。
