MonitorGetName

指定されたモニターのOS上の名前を返します。

Name := MonitorGetName(N)

パラメータ

N

Type: Integer

省略されたときは、プライマリモニタが使用されます。そうでないときは、1からMonitorGetCount関数で返された数までのモニターナンバーを指定します。

戻り値

Type: String

この関数は、指定されたモニターのオペレーティングシステム名を返します。

エラー処理

失敗した場合は例外がスローされます。

関連

SysGet, Monitor functions

例

この関数の実例は、「Monitor関数」ページの例1をご覧ください。
