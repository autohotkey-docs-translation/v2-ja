MonitorGetName

指定されたモニタのオペレーティングシステム名を返します。

Name := MonitorGetName(N)

パラメータ

N

型：整数

モニター番号。1からMonitorGetCountによって返される番号の間。省略された場合は、プライマリモニタが使用されます。

戻り値

型：文字列

この関数は、指定されたモニターのオペレーティングシステム名を返します。

エラー処理

失敗した場合は例外が発生します。

関連

SysGet, Monitor functions

例

この機能のデモンストレーションは、「モニター機能」ページの例1を参照してください。
