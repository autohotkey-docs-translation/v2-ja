GetKeyName

Retrieves the name/text of a key.

Name := GetKeyName(KeyName)

パラメータ

KeyName

型：String

キーボードの任意の1文字、またはキーリストのキー名のいずれかを指定することができます。例：B, 5, LWin, RControl, Alt, Enter, Escape.

あるいは、vkFFのような明示的な仮想鍵コード、sc01Dのような明示的なスキャンコード、vk1Bsc001のようなVKとSCの組み合わせ（この順）であってもよい。なお、これらのコードは16進数である必要があります。

戻り値

型：String

この関数は、指定されたキーの名前を返し、キーが無効または無名である場合は空白を返す。

関連

GetKeyVK, GetKeySC, GetKeyState, Key List

例

Escの英語名を取得し、報告します。

MsgBox GetKeyName("Esc") ; Shows Escape
MsgBox GetKeyName("vk1B") ; Shows also Escape
