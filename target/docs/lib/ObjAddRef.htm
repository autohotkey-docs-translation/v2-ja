ObjAddRef / ObjRelease

オブジェクトの参照カウントを増減させる。

NewRefCount := ObjAddRef(Ptr)
NewRefCount := ObjRelease(Ptr)

パラメータ

Ptr

型：整数

アンマネージドオブジェクトポインタまたはCOMインターフェイスポインタです。

戻り値

型：整数

これらの関数は、新しい参照カウントを返します。この値は、デバッグの目的でのみ使用する必要があります。
