Break

あらゆるタイプのループ文を終了（終了）させる。

Break LoopLabel

パラメータ

LoopLabel

省略時または1の場合、この文が囲まれている最も内側のループに適用されます。Otherwise, specify which loop this statement should apply to; either by label name or numeric nesting level. If a label is specified, it must point directly at a loop statement.

LoopLabelは定数値でなければならない。変数や式はサポートされていない。ただし、括弧で囲まれた単一のリテラル数値または引用文字列は例外です。事例：break("outer")

備考

BreakとContinueの使用は、通常、スクリプトをより読みやすく、保守的にするため、Gotoよりも推奨されています。

関連

Continue, Loop, While-loop, For-loop, Blocks, Labels

例

varが 25 より大きい場合、ループを中断します。

Loop
{
    ; ...
    if (var > 25)
        break
    ; ...
    if (var <= 5)
        continue
}

ネストしたループの中から外側のループをブレイクします。

outer:
Loop 3
{
    x := A_Index
    Loop 3
    {
        if (x*A_Index = 6)
            break outer  ; ブレーク2またはgoto break_outerに相当します。
        MsgBox x "," A_Index
    }
}
