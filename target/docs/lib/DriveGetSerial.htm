DriveGetSerial

指定されたドライブのボリュームシリアルナンバーを返します。

Serial := DriveGetSerial(Drive)

パラメータ

Drive

Type: String

ドライブレターの後にコロンとオプションのバックスラッシュを付けたもの、または\server1\share1のようなUNC名を指定します。

戻り値

Type: Integer

この関数はドライブのボリュームシリアルナンバーを返します。See Format for how to convert it to hexadecimal.

エラー処理

失敗した場合は例外が発生します。
