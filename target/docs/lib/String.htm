String

値を文字列に変換します。

StrValue := String(Value)

戻り値

型：文字列

この関数は、Valueを文字列に変換した結果、またはValueが文字列の場合はValueそのものを返します。

Valueが数値の場合、デフォルトの10進数書式が使用されます。

Valueがオブジェクトの場合は、Value.ToString() を呼び出した結果が返り値となります。オブジェクトにそのようなメソッドがない場合、MethodErrorがスローされます。このメソッドはデフォルトでは実装されていないので、スクリプトで定義する必要があります。
