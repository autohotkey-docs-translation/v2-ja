SoundGetName

サウンドデバイスやコンポーネントの名前を取得します。

Name := SoundGetName(Component, Device)

パラメータ

コンポーネント

型：文字列または整数

コンポーネントの表示名とインデックス。例えば、1、"Line in"または"Line in:2"。

省略または空白の場合、デバイス自体の名前が取得される。

詳しくは「コンポーネント（サウンド機能）」をご覧ください。

Device

型：文字列または整数

デバイスの表示名とインデックス。例えば、1、"Speakers"、"Speakers:2"、"Speakers (Example HD Audio)"などです。

このパラメーターが省略された場合、デフォルトはシステムのデフォルト再生デバイス（デバイス1とは限らない）となる。

詳しくはデバイス（サウンド機能）をご覧ください。

戻り値

型：文字列

デバイスまたはコンポーネントの名前。

エラー処理

失敗すると以下の例外のいずれかがスローされる：

エラークラスメッセージ
TargetErrorデバイスが見つかりません
コンポーネントが見つかりません
OSErrorシステムが生成する。

関連

サウンド機能

例

デフォルトの再生デバイス名を取得し、報告する。

default_device := SoundGetName()
MsgBox 'The default playback device is "' default_device '".'

最初のデバイスの名前を取得し、報告する。

device1 := SoundGetName( , 1)
MsgBox 'Device 1 is "' device1 '".'

最初のコンポーネントの名前を取得して報告する。

component1 := SoundGetName(1)
MsgBox 'Component 1 is "' component1 '".'

より複雑な例については、サウンドカード解析スクリプトを参照のこと。
